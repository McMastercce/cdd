Dropbox

We use Dropbox to share files. The following table lists the files that you will use for integration located in the Dropbox/CCE Development - Templates/6. LST Templates folder. If you have not been added to our Dropbox contact an LST.

The files in the Dropbox get updated frequently. Always grab a new copy of these files before beginning integration of a course.

Location Purpose
Default Turnitin Settings.docx Contains the standard Turnitin settings that we configure for all Assignment folders. If you did not receive any specific configuration settings for Turnitin in your course, refer to this document and deafult to these settings.
Master QA Checklist.xlsx Contains a checklist of items to complete when integrating a course.
Template Changes Log.xlsx Documentation of updates to our CCE Template in Avenue. Refer to this near the end of integration to see if any of the pages have been udpate. If you are unsure weather or not you should update any
Tool Settings.docx Template for documenting tool settings that need to be configured in the course. Use this when you meet with the ID to discuss tool settings.
TEMPLATE_101 Contains a folder titled Avenue with a copy of the CCE Course Template content files in Avenue.
While you can use these files as a starting point to develop webpages, it is reccomended that you copy all components from the CCE Template in Avenue to your course using the Import/Export/Copy feature. This will ensure that settings are copied and configured correctly in your course.
